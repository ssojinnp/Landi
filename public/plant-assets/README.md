# Plant asset guide

Use transparent PNG cutouts for plant symbols. Assets should be top-view, centered, and cropped with a small transparent margin so they align naturally on the board.

## Naming

- Canopy trees: `tree-canopy-01.png`, `tree-canopy-02.png`, `tree-canopy-03.png`
- Conifers: `tree-conifer-01.png`, `tree-conifer-02.png`
- Shrubs: `shrub-01.png`, `shrub-02.png`
- Flowering shrubs: `shrub-flower-01.png`
- Variegated shrubs: `shrub-variegated-01.png`
- Groundcovers: `groundcover-01.png`, `groundcover-02.png`
- Upright grass tufts: `groundcover-tuft-01.png`
- Broadleaf groundcovers: `groundcover-broadleaf-01.png`
- Flower beds: `flower-bed-01.png`, `flower-bed-02.png`

## Export Rules

- PNG with transparency.
- Square canvas is preferred.
- Keep the plant visually centered.
- Avoid baked-in dark backgrounds or checkerboard backgrounds.
- Keep contact shadow subtle. The app already adds a small board shadow.
